## What I have learned

- There is an easy way to wrap elements in html files inside vscode. Mark the lines that you want to wrap and then to `CTRL - SHIFT - P` and write `Emmet` and choose the first option.

- Emmet is a useful tool to use inside vscode. Instead of writing tags one after the other you can do write a sequence of emmet code and it will generate html for you. [Cheatsheet](https://docs.emmet.io/cheat-sheet/)
